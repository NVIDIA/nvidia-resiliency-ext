# Corpus Labels

This directory contains corpus-facing support only; the repository does not
embed source logs or define a second case format.

- [REQUIREMENTS.md](../docs/REQUIREMENTS.md) owns corpus and three-root isolation requirements.
- [SCHEMA.md](../docs/SCHEMA.md) owns the mirrored `gold.json` path and label schema.
- [TUNING.md](../docs/TUNING.md) owns review and approval workflow.
- [the harness README](../README.md) owns executable one-log and corpus commands.
